# Gestor de Inventario Qt

Sistema de inventario desarrollado en C++ utilizando Qt y SQLite.

## Características

- Registro de artículos
- Edición y eliminación
- Búsqueda dinámica
- Alertas de stock bajo
- Exportación CSV
- Interfaz gráfica moderna
- Persistencia con SQLite

## Tecnologías utilizadas

- C++
- Qt Widgets
- SQLite
- CMake
- GitHub
